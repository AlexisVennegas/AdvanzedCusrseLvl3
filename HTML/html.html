Enlaces

usamos la etiqueta a 
<a href="http://www.google.com">Google</a>

Formularios

usamos la etiqueta form
el method puede ser get o post, se usa post para enviar datos sensibles
y el action es la url a la que se envian los datos 
<form action="procesar.php" method="post"></form>

Inputs

sirven para introducir datos 
<input type="text" name="nombre" id="nombre"></input>
Label es el texto que aparece al lado del input y el for es el id del input 
<label for="nombre">Nombre</label>
