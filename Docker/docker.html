dockerHub

dockerhub es un repositorio de imagenes de docker, en el cual se pueden encontrar imagenes de todo tipo, desde imagenes de sistemas operativos, hasta imagenes de aplicaciones, como por ejemplo wordpress, mysql, etc.

docker deamon

El daemon de docker es un proceso que corre en segundo plano en el sistema operativo, el cual se encarga de administrar los contenedores de docker, asi como tambien de las imagenes de docker.

docker file

Un dockerfile es un archivo de texto que contiene una serie de instrucciones que se utilizan para crear una imagen de docker. Estas instrucciones pueden ser desde instalar paquetes, copiar archivos, configurar variables de entorno, etc.
